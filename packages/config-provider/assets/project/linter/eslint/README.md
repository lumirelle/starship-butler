# ESLint Configs

This folder contains four sub-folders which are categorized based on your project type:

- Library: For JavaScript/TypeScript library projects
- Vue 2: For Vue 2 projects & Nuxt 2 projects + UnoCSS
- Vue 3: For Vue 3 projects + UnoCSS
- Nuxt 4: For Nuxt 4 projects + UnoCSS, including more Nuxt specific rules than Vue 3 config

> [!Note]
>
> Nuxt 2 projects should use the Vue 2 config. Nuxt 3 projects are encouraged to upgrade to Nuxt 4.
