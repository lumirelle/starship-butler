# ESLint Configs

This folder contains four sub-folders which are categorized based on your project type:

- JavaScript: For JavaScript/TypeScript library projects
- Vue 2: For Vue 2 projects & Nuxt 2 projects
- Vue 3: For Vue 3 projects
- Nuxt 3+: For Nuxt 3+ projects, including more Nuxt specific rules than Vue 3 config
