# TypeScript Configs

This folder contains four sub-folders categorized by project type:

- Library: For TypeScript library projects
- Vue + TypeScript: For Vue application projects
- Vue + JavaScript: It's recommended to use `// @ts-check` to migrate to TypeScript gradually
- Nuxt 4: For Nuxt 4 projects

> [!Note]
>
> Nuxt 2 projects may reuse the Vue + JavaScript configuration, with custom "compilerOptions.types" options.
>
> Nuxt 3 projects are encouraged to upgrade to Nuxt 4.
