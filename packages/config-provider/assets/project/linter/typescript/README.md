# TypeScript Configs

This folder contains four sub-folders categorized by project type:

- Library: For TypeScript library projects
  - Singlerepo: For TypeScript single-repo  projects
  - Monorepo: For TypeScript monorepo projects
- App: For TypeScript application projects
  - Vue 3 + TypeScript: For TypeScript Vue projects
  - Nuxt 4: For TypeScript Nuxt projects
